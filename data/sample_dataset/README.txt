Sample Dataset for Quick Testing
==================================

This is a small subset of the HAM10000 dataset for quick testing and demonstration.

Total Images: 70
Images per class: ~10

Class Distribution:
  - akiec: 10 images
  - bcc: 10 images
  - bkl: 10 images
  - df: 10 images
  - mel: 10 images
  - nv: 10 images
  - vasc: 10 images

Usage:
------
This sample dataset can be used to:
1. Quickly test the code without downloading the full 2GB dataset
2. Verify the pipeline works on your system
3. Debug or modify code with faster iteration

Note: Results from this sample will differ from full dataset results
      due to the small sample size and reduced training data.

To use the full dataset, follow instructions in data_readme.txt
